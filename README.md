# WarsInReverie.github.io
Website for the "Wars in Reverie" series.

[WarsInReverie.github.io](https://warsinreverie.github.io/)

This website was built with Hugo, and uses the Blowfish theme (to be modified)

Inspired by millmint.net.

### Author data

Default author parameters stored in config/_default/languages.em.toml

Specific author data stored in data/authors

### Posts data

Posts are contained in the content/ folder, and seperated into different catagories for post type.
